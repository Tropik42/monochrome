## Description

## Install and run

Установка и запуск

`npm i` - установка npm-зависимостей  
`npm run dev` - сборка и запуск приложения на локальном хосте  
`npm run build` - сборка приложения  
`npm run start` - сборка приложения и запуск сервера на порту 3000
